# THOUGHT Signing OS Pack

Run ID: `sepolia-thought-signing-os-pack-20260515T050730Z`

Standalone Sepolia deploy pack for THOUGHT. It deploys THOUGHT contracts, registers `THOUGHT.v1.md`, configures the existing PATH movement `THOUGHT` to the deployed `ThoughtNFT`, and freezes that PATH movement config.

This pack is designed for Signing OS. It does not require a git checkout or npm install on Signing OS.

The `source/` directory is a curated deploy source snapshot from the exact source commit, not a full repository archive. See `PACK-MANIFEST.json.source_snapshot` for the included paths.

Use `RUNBOOK.md` for the operator sequence.
